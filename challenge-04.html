SOC Analyst Briefing
EDR flagged this executable on workstation CN-WS-047 at 08:14 UTC. The file was delivered via a phishing email with the subject line "Q4 Invoice — Action Required". Static analysis confirms it is a .NET-based Remote Access Trojan capable of keylogging, credential theft, and persistent outbound C2 communication.

The malware is actively beaconing to an external IP address. Your task is to identify the C2 server IP so the network team can block it immediately.
